# **Laptop** :computer:

## Tecnologías Utilizadas

* Java
* Spring
* Swagger
* API REST
* Tests unitarios


## Instalación

Pasos a seguir:

1. Clonar el repositorio:

```
git clone https://github.com/sharonOstrovsky/Laptop.git
```

2. Corra la aplicación
3. Ingrese al local host:

```
http://localhost:8080//swagger-ui.html
```
